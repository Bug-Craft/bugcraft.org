### Java 版本

ㅤㅤ`play.bugcraft.org` 或者 `host.bugcraft.org:55565`

ㅤㅤ适用于 1.8.0 - 1.20.1 的 Minecraft Java Edition。

### Bedrock 版本

* `host.bugcraft.org`

ㅤㅤ这是服务器的地址。接下来，根据版本进行判断要使用的端口号。

`29132`

`39132`

ㅤㅤ适用于 1.19 - 1.20 的 Minecraft Bedrock Edition！允许你以 Bedrock 版本玩家的身份进行登录。
